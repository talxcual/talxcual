## 🚀 Featured Projects

### 🌎 Environmental Monitoring Station with Applied AI
- ESP32-based environmental monitoring system.
- Real-time acquisition of CO₂, TVOC, temperature and humidity data.
- MQTT communication and remote management application.
- Multilayer neural network for intelligent environmental data analysis.

### 📱 ArriendApp
- Native Android application developed with Kotlin and Android Studio.
- MVVM architecture and Firebase integration.
- NoSQL persistence and real-time data synchronization.
- AI-powered assistant for automating operational tasks.

### 🎮 Emoji Breakout
- Native Android game developed with Kotlin.
- Firebase Authentication and NoSQL backend.
- Global ranked system with real-time synchronization.
- Persistent user profiles and score management.

### 🎮 Not Forest
- 3D Survival Horror game developed with Unity and C#.
- Inventory and combat systems.
- Enemy AI and dynamic environments.
- Graphics optimization using Universal Render Pipeline (URP).
